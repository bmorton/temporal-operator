# TemporalDevServer Server-Version Mapping — Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Make `TemporalDevServer.spec.version` accept a Temporal **server** version (e.g. `1.31.1`) and map it to the correct `temporalio/temporal` CLI image via the version matrix, defaulting to the latest supported server version when omitted.

**Architecture:** Add a per-server-line `devServerCLIVersion` to the generated version matrix and two resolvers in `internal/temporal`. Make `internal/resources/devserver.go` resolve the image from the server version (with an error on unsupported versions), keep `spec.image` as an escape hatch, validate in the controller (failed condition, no webhook), and update all merged examples/sample/e2e/docs that used the old CLI-tag semantics.

**Tech Stack:** Go 1.26, kubebuilder/controller-runtime, `make gen-version-matrix` / `make manifests`, `make test` (envtest).

**Spec:** `docs/superpowers/specs/2026-06-19-devserver-server-version-design.md`

```go
// DevServerCLIVersion returns the temporalio/temporal CLI image tag that runs
// the given Temporal server version, or an empty string when the server version
// is unknown or has no dev-server mapping.
func DevServerCLIVersion(serverVersion string) string {
	info, err := LookupVersion(serverVersion)
	if err != nil {
		return ""
	}
	return info.DevServerCLIVersion
}

// LatestSupportedVersion returns the highest supported Temporal server patch
// version in the matrix, or an empty string when the matrix is empty.
func LatestSupportedVersion() string {
	vs := SupportedVersions()
	if len(vs) == 0 {
		return ""
	}
	return vs[len(vs)-1] // SupportedVersions() is sorted ascending by semver.
}
```

2. Replace `DevServerImage` and add `DevServerServerVersion`:

```go
// DevServerImage returns the container image for a dev server. When Image is set
// it is used verbatim; otherwise the Temporal server Version (or the latest
// supported version when empty) is mapped to the matching temporalio/temporal
// CLI image. It returns an error when the server version is unsupported.
func DevServerImage(dev *temporalv1alpha1.TemporalDevServer) (string, error) {
	if dev.Spec.Image != "" {
		return dev.Spec.Image, nil
	}
	version := dev.Spec.Version
	if version == "" {
		version = temporal.LatestSupportedVersion()
	}
	cli := temporal.DevServerCLIVersion(version)
	if cli == "" {
		return "", fmt.Errorf("unsupported dev server version %q: not in the supported matrix %v",
			version, temporal.SupportedVersions())
	}
	return "temporalio/temporal:" + cli, nil
}

// DevServerServerVersion returns the Temporal server version reported in status:
// the explicit Version, the latest supported version when Version is empty, or
// an empty string when a raw Image override is used.
func DevServerServerVersion(dev *temporalv1alpha1.TemporalDevServer) string {
	if dev.Spec.Version != "" {
		return dev.Spec.Version
	}
	if dev.Spec.Image != "" {
		return ""
	}
	return temporal.LatestSupportedVersion()
}
```
